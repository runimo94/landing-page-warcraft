# LandingPageWarcraft

Este proyecto es una replica de una "landing page" desarrollada con Angular y utilizando el framework de estilos CSS Tailwind. La página está inspirada en el videojuego World of Warcraft y cuenta con algunos toques temáticos del juego. El objetivo de este proyecto es practicar mis habilidades en el desarrollo web con estas herramientas.
